# Kanban Pro

Real-time collaborative Kanban. Notion-inspired warm white/brown palette.

## Start

```bash
# Terminal 1
cd server && npm install && npm start

# Terminal 2
cd client && npm install && npm run dev
```

Open http://localhost:3000

## Features
- Landing page with board preview
- Create board or join with room code (Enter key works)
- Invite modal with big room code + copy link button
- Warm white/brown Notion-style UI
- Dark / light mode
- Real-time drag-and-drop with spring animation across all columns
- Full audit log panel with before/after diffs
- WIP limits, task pinning, archiving, duplication
- Checklists, due dates, labels, assignees, time estimates
- Comments with typing indicators and delete
- Live presence — see who's viewing which task
- Members panel with activity feed
- Stats bar with progress
- Conflict resolution with optimistic rollback
- Cmd+K search palette
